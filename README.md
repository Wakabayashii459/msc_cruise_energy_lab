# MSC Cruise Energy Lab

A WSL-based project for learning cruise-vessel energy analytics, machine learning,
forecasting, optimization, and reporting.

## Core themes

- fuel consumption analytics
- hull and engine performance
- SFOC
- propulsive vs non-propulsive efficiency
- shore power analytics
- emissions and sustainability reporting
- fleet-scale data engineering
- Python + SQL + Power BI-ready outputs

## Planned phases

1. Foundation
2. Synthetic cruise-fleet data generation
3. Warehouse and SQL models
4. KPI analytics
5. Machine learning models
6. Optimization and reporting
7. API and dashboard exports
